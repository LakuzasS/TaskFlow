API Modules

Liste des modules de l’API :

api.api_google.oauth2callback()[source]

Route callback pour gérer le retour d’authentification OAuth 2.0 de Google.

Cette fonction récupère le code d’authentification retourné par Google, échange ce code contre un token, et valide le token reçu pour obtenir les informations de l’utilisateur (email et nom). Ensuite, elle tente de connecter l’utilisateur à l’application via la fonction login_with_google. Si la connexion réussit, elle envoie un message au système de file d’attente pour notifier l’application du succès de la connexion.

Returns:

Une réponse HTTP avec un message de succès ou d’erreur.

Return type:

Response

api.api_google.google_login()[source]

Route API pour effectuer une connexion avec un token ID Google.

Cette fonction prend un id_token envoyé par l’utilisateur dans le corps de la requête, tente de l’utiliser pour connecter l’utilisateur via la fonction login_with_google.

Parameters:

id_token (str) – Le token ID Google fourni par l’utilisateur.

Returns:

Une réponse HTTP avec un message de succès ou d’erreur en fonction de la connexion.

Return type:

Response
